# Projeto_SAP_ROI
# SAP RAP Training ROI

This repository contains a training project developed to study and demonstrate the **SAP RESTful ABAP Programming Model (RAP)** in the **ABAP Cloud environment**.

The goal of this project is to build a simple **Human Resources application** capable of tracking employee training programs and analyzing the **Return on Investment (ROI)** of training initiatives.

The project focuses on learning and applying the core components of RAP development, including:

- ABAP Cloud development
- CDS Data Models
- Behavior Definitions
- Business Services
- OData exposure
- Clean architecture for SAP applications

## Technologies

- SAP BTP (Business Technology Platform)
- ABAP Cloud
- SAP RAP (RESTful ABAP Programming Model)
- ABAP Development Tools (ADT) in Eclipse
- GitHub for version control

## Project Structure

The repository is organized following the RAP layered architecture:


ZHR_TRAINING_ROI
│
├── Data Model (Tables)
├── CDS Views
├── Behavior Definitions
├── Service Definitions
└── Service Bindings


## Purpose

This project is intended for:

- Learning SAP RAP development
- Demonstrating ABAP Cloud capabilities
- Serving as a reference implementation for beginners
